01 Eje tonal — el trabajo vive en el rango

No un punto.
Un rango.

La identidad no se fija en un solo tono: se mueve entre lo frío/generativo (paramétrico, glitch, forma algorítmica) y lo cálido/atmosférico (niebla, sobrecrecimiento, lo orgánico). Mueve el control y el sistema entero se reacomoda: acento, color terciario y la capa de vegetación que invade los bordes.

Decisión de esta sesión: tensión frío + orgánico activa. Lo cálido no es decorativo, entra por los bordes.

06 Aplicación — el sistema instanciado en el deck

Un cyberdeck
con identidad.

El sistema no es un tema descargado: son tokens que viajan de Figma a archivos de config (CSS real, dotfiles versionados en Git como source of truth). El deck sirve para todo —terminal de campo, máquina de escritura, servidor local, herramienta contemplativa— y la UI cambia de capa según el uso, manteniendo la misma identidad.

Componente "ventana" que simula el OS — gap del sistema, ahora prototipado →
